# คู่มือการใช้งานโปรแกรม PDF Footer Image Adder

โปรแกรมนี้ถูกสร้างขึ้นเพื่อช่วยให้คุณสามารถนำรูปภาพต่างๆ (เช่น ลายเซ็น หรือโลโก้) ไปวางลงบนทุกหน้าของไฟล์ PDF หลายๆ ไฟล์ได้อย่างอัตโนมัติ โดยสามารถปรับขนาด (Scale), ความโปร่งใส (Opacity) และตำแหน่งพิกัด (X, Y) ได้อย่างอิสระสำหรับแต่ละรูปภาพ

## 📂 โครงสร้างโฟลเดอร์และไฟล์ที่สำคัญ

ภายในโฟลเดอร์ `D:\dev\MTAddImgToPdf\my_workspace\pdf\` จะประกอบไปด้วยไฟล์และโฟลเดอร์ดังนี้:

- **`source\`**: โฟลเดอร์สำหรับนำไฟล์ PDF ต้นฉบับที่คุณต้องการจะประทับตรา/ลายเซ็นมาวางไว้
- **`output\`**: โฟลเดอร์สำหรับเก็บไฟล์ PDF ผลลัพธ์ที่โปรแกรมทำงานเสร็จแล้ว
- **`config.toml`**: ไฟล์สำหรับตั้งค่าโปรแกรมทั้งหมด (เช่น ระบุรูปภาพ ขนาด ตำแหน่ง)
- **`run_add_footer.bat`**: ไฟล์สำหรับกดรันโปรแกรม (ดับเบิลคลิกเพื่อเริ่มทำงาน)
- **`add_footer.py`**: ไฟล์ซอร์สโค้ดภาษา Python ที่เป็นหัวใจหลักในการทำงาน

---

## ⚙️ วิธีการตั้งค่าในไฟล์ `config.toml`

คุณสามารถเปิดไฟล์ `config.toml` ด้วย Notepad หรือโปรแกรม Text Editor ทั่วไป เพื่อแก้ไขการตั้งค่า โดยแบ่งออกเป็น 2 ส่วนหลัก:

### 1. ส่วนกำหนดที่อยู่โฟลเดอร์ (`[paths]`)
```toml
[paths]
source_dir = "D:\\dev\\MTAddImgToPdf\\my_workspace\\pdf\\source"
output_dir = "D:\\dev\\MTAddImgToPdf\\my_workspace\\pdf\\output"
```
*แนะนำให้คงค่าเดิมไว้ ยกเว้นต้องการย้ายโฟลเดอร์ไปที่อื่น (ระวังการใช้เครื่องหมาย `\\` ในการคั่นโฟลเดอร์ด้วย)*

### 2. ส่วนกำหนดรูปภาพ (`[[images]]`)
คุณสามารถเพิ่มรูปภาพลงไปกี่รูปก็ได้ โดยการคัดลอกบล็อก `[[images]]` เพิ่มลงไปด้านล่าง ตัวอย่าง:

```toml
[[images]]
path = "D:\\dev\\MTAddImgToPdf\\my_workspace\\pdf\\sign\\p6-coop-logo.png"
scale_percent = 10
opacity = 0.3
pos_x = 100
pos_y = 150
```

**คำอธิบายค่าต่างๆ:**
- **`path`**: ที่อยู่ไฟล์รูปภาพที่ต้องการนำมาประทับ (รองรับ PNG, JPG)
- **`scale_percent`**: ปรับย่อ/ขยายรูปภาพ คิดเป็นเปอร์เซ็นต์เทียบกับรูปต้นฉบับ 
  - `100` = ขนาดเท่าต้นฉบับ
  - `50` = ขนาดเล็กลงครึ่งหนึ่ง
- **`opacity`**: ระดับความโปร่งใสของรูปภาพ
  - `1.0` = ทึบแสง 100% (มองไม่ทะลุ)
  - `0.0` = โปร่งใส 100% (มองไม่เห็นรูปเลย)
  - ค่าระหว่างนั้น (เช่น `0.5`, `0.8`) คือระดับความโปร่งใสตามลำดับ
- **`pos_x`**: ตำแหน่งพิกัดในแนวนอน (แกน X) ยิ่งค่ามาก รูปภาพจะยิ่งขยับไปทาง **ขวา**
- **`pos_y`**: ตำแหน่งพิกัดในแนวตั้ง (แกน Y) ยิ่งค่ามาก รูปภาพจะยิ่งขยับ **ลงล่าง**
  *(ข้อควรระวัง: กระดาษแนวนอน (Landscape) จะมีความสูงประมาณ 612 ดังนั้น pos_y ไม่ควรเกินค่าความสูงของกระดาษ มิฉะนั้นรูปจะหลุดออกนอกหน้ากระดาษ)*

## 🚀 วิธีการใช้งานโปรแกรม

1. นำไฟล์ PDF ทั้งหมดที่ต้องการประทับรูปภาพ ไปใส่ไว้ในโฟลเดอร์ **`source`**
2. ตรวจสอบหรือแก้ไขการตั้งค่ารูปภาพในไฟล์ **`config.toml`** (แก้ไขตำแหน่ง ย่อขยาย ความโปร่งใส)
3. ดับเบิลคลิกที่ไฟล์ **`run_add_footer.bat`**
4. จะมีหน้าต่างสีดำ (Terminal) เด้งขึ้นมา รอจนกว่าโปรแกรมจะประมวลผลเสร็จและขึ้นข้อความว่า *"เสร็จสิ้นการทำงานทั้งหมด!"*
5. กดปุ่มใดๆ เพื่อปิดหน้าต่าง แล้วเข้าไปรับไฟล์ PDF สำเร็จรูปได้ที่โฟลเดอร์ **`output`**

## ⚠️ การแก้ปัญหาเบื้องต้น (Troubleshooting)

**ปัญหา: โปรแกรมแจ้งเตือนว่า `Permission denied`**
> **สาเหตุ:** คุณกำลังเปิดไฟล์ PDF ในโฟลเดอร์ `output` ค้างไว้ด้วยโปรแกรมอื่น (เช่น Adobe Acrobat, Edge หรือ Chrome) โปรแกรมจึงไม่สามารถเขียนไฟล์ทับลงไปได้  
> **วิธีแก้:** ปิดไฟล์ PDF นั้นให้เรียบร้อยก่อน แล้วจึงกดรัน `run_add_footer.bat` ใหม่อีกครั้ง

**ปัญหา: โปรแกรมแจ้งเตือนว่า `Warning: ไม่พบไฟล์รูปภาพที่ ...`**
> **สาเหตุ:** พิมพ์ชื่อไฟล์หรือที่อยู่ไฟล์รูปภาพใน `config.toml` ผิด หรือไม่มีไฟล์รูปภาพนั้นอยู่จริง  
> **วิธีแก้:** เข้าไปตรวจสอบชื่อไฟล์และแก้ไข `path` ใน `config.toml` ให้ถูกต้อง

**ปัญหา: รูปภาพไม่ปรากฏในหน้า PDF**
> **สาเหตุ:** ตำแหน่ง `pos_x` หรือ `pos_y` ที่ตั้งค่าไว้น่าจะอยู่นอกพื้นที่ของกระดาษ  
> **วิธีแก้:** ลองปรับลดค่า `pos_x` และ `pos_y` ให้มีค่าน้อยลง (เพื่อขยับรูปขึ้นและไปทางซ้าย)
